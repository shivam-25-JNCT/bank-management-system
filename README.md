# 🏦 Bank Management System (Console-Based)

A simple **Bank Management System** built using **Python** and **SQLite**. This project is a console-based application that demonstrates the implementation of **Object-Oriented Programming (OOP)** concepts along with database operations.

## 📌 About the Project

This project simulates the basic functionalities of a banking system. Users can create accounts, deposit and withdraw money, transfer funds, check account balances, update their profile, and view transaction history.

The project was built to strengthen my understanding of **Python, OOP principles, SQLite database management, and CRUD operations**.

---

## 🚀 Features

* 🔐 User Registration & Login
* 💰 Deposit Money
* 💸 Withdraw Money
* 💳 Transfer Money Between Accounts
* 📊 Check Account Balance
* 📜 View Last 5 Transaction History
* 🔑 Change Password
* 👤 Update User Profile
* 💾 Data Stored Using SQLite Database
